# Usage
<!-- usage -->
```sh-session
$ npm install -g @sapling/cli
$ sapling COMMAND
running command...
$ sapling (-v|--version|version)
@sapling/cli/0.1.0 darwin-x64 node-v15.2.0
$ sapling --help [COMMAND]
USAGE
  $ sapling COMMAND
...
```

OPTIONS
  --all  see all commands in CLI
